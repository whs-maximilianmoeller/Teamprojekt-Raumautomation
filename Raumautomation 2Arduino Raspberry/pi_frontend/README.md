# Raspberry Pi Frontend (Web Dashboard)

**Funktion:**
Benutzeroberfläche zur Visualisierung und Steuerung.

**Technologie:**
- HTML/CSS/JavaScript (oder Framework nach Wahl)
- Chart.js (für Graphen)

**Aufgaben:**
- Visuelle Darstellung der Temperatur- und Feuchtigkeitswerte.
- Manuelle Steuerungsmöglichkeiten (z.B. Lüfter an/aus, Sollwert-Vorgabe).
- Systemstatus-Anzeige.
